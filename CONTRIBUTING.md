# Contributing to Saola CLI

## Development

- Go 1.26+ (see go.mod)
- make build, make test, make lint
- See [OpenSaola CONTRIBUTING guide](https://gitee.com/opensaola/opensaola/blob/master/CONTRIBUTING.md) for detailed architecture and workflow

## Code Style
- go fmt ./...
- Bilingual comments (English above, Chinese below)
- Use lang.T(zh, en) for user-facing strings
