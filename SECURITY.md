# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Saola CLI, please report it
responsibly by emailing **security@opensaola.io**.

Please do **not** open a public issue for security vulnerabilities.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

We will acknowledge your report within 48 hours and aim to provide a fix
or mitigation within 7 days for critical issues.

## Supported Versions

Security updates are provided for the latest minor release only.
